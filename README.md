# Hướng Dẫn Cài Đặt và Sử Dụng UTC Grade Export

## 📋 Giới Thiệu
UTC Grade Export là một tiện ích mở rộng (extension) cho Chrome giúp bạn xuất bảng điểm từ hệ thống tra cứu điểm UTC thành file PDF một cách dễ dàng.

---

## 🔧 Hướng Dẫn Cài Đặt

### Bước 1: Tải Tệp Tiện Ích
- Tải xuống toàn bộ thư mục chứa tiện ích
- Hoặc clone từ repository nếu có

### Bước 2: Mở Chrome
- Khởi động trình duyệt Google Chrome

### Bước 3: Truy Cập Trang Quản Lý Extension
- Gõ vào thanh địa chỉ: `chrome://extensions/`
- Hoặc đi vào **Menu (≡) → Thêm công cụ → Tiện ích**

### Bước 4: Bật Chế Độ Nhà Phát Triển
- Tìm và bật nút **"Chế độ dành cho nhà phát triển"** ở góc trên bên phải trang

### Bước 5: Tải Tiện Ích Chưa Đóng Gói
- Nhấp nút **"Tải tiện ích đã giải nén"**
- Chọn thư mục chứa extension (thư mục chứa file `manifest.json`)

### Bước 6: Xác Nhận Cài Đặt
- Tiện ích sẽ xuất hiện trong danh sách các extension đã cài
- Kiểm tra xem biểu tượng tiện ích đã xuất hiện trên thanh công cụ Chrome chưa

---

## 📖 Hướng Dẫn Sử Dụng

### Bước 1: Truy Cập Hệ Thống UTC
- Đăng nhập vào trang tra cứu điểm: **https://qldt.utc.edu.vn/congthongtin/**
- Đăng nhập bằng tài khoản sinh viên của bạn

### Bước 2: Tra Cứu Bảng Điểm
- Chọn học kỳ hoặc năm học muốn xem điểm
- Chờ bảng điểm tải xong

### Bước 3: Xuất Bảng Điểm
- Sau khi bảng điểm hiển thị, tiện ích sẽ tự động thêm nút **"Xuất PDF"** 
- Nhấp vào nút này để tạo file PDF

### Bước 4: Lưu File
- Chọn vị trí lưu file trên máy tính
- Đặt tên cho file PDF (ví dụ: "Bang_diem_HK1_2024")
- Nhấp **"Lưu"**

✅ Hoạt động trực tiếp trên trang tra cứu điểm UTC

## 🐛 Khắc Phục Sự Cố

### Tiện ích không hiển thị nút xuất PDF
- **Kiểm tra:** Bạn đã đăng nhập vào qldt.utc.edu.vn chưa?
- **Giải pháp:** Đảm bảo đã truy cập đúng URL: `https://qldt.utc.edu.vn/congthongtin/`
- **Cách khác:** Bấm F5 để tải lại trang

### Tải extension thất bại
- Kiểm tra lại file `manifest.json` có trong thư mục không
- Thử bỏ cache browser: Nhấp **Ctrl+Shift+Delete** (hoặc **⌘+Shift+Delete** trên Mac)
- Cố gắng tải lại extension

### PDF không xuất được
- Đảm bảo trình duyệt đã cho phép tải PDF
- Kiểm tra không gian lưu trữ máy tính có đủ không
- Thử xuất lại với một tên file khác

## ⚙️ Yêu Cầu Hệ Thống

- **Truy cập:** Tài khoản sinh viên UTC hợp lệ
